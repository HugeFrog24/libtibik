# libtibik

Release distribution repository for Tibik.

> [!NOTE]
> libtibik is entering a LTS / maintenance phase. Bug fixes and critical compatibility updates may still land, but my primary focus is shifting toward the more modern and feature-complete libTSM ([ThatSkyMod](https://github.com/XeTrinityz/ThatSkyMod-Android)), where major new development will happen. Switching to that is strongly recommended.

## Roadmap
